Разработка тем для Wordpress

Перевод 26.02.2014 10:12

Введение

Это статья о разработке тем для Wordpress. Если вы хотите узнать больше о том, как установить и использовать темы, почитайте в кодексе раздел Using Themes. Этот топик отличается от Using Themes, потому как тут обсуждаются технические аспекты написания кода для построения вашей собственной темы,а не как активировать тему или где ее можно получить.

Почему Wordpress Темы ?

Worpdress темы - это файлы, которые работают вместе, создавая дизайн и функционал вордпресс сайта.Каждая тема может быть различной, предлагающей множество вариантов владельцам сайта для мгновенного изменения внешнего вида.

Вы желаете разработать тему для собственного сайта, для проекта клиента или для отправки в директорию тем WordPress. Почему еще вы должны разрабатывать темы WordPress ?

  • Для создания уникального вида вашего веб-сайта
  • Оценить преимущество шаблонов, тегов шаблонов и цикла WordPress генерировать различные страницы и виды страниц
  • Обеспечить альтернативные шаблоны для специфичечких особенностей сайта, таких как страницы рубрик и страницы результатов поиска
  • Для быстрого переключения между двумя слоями сайта или оценки преимущества переключателя стилей тем, чтобы разрешить владельцам сайтов менять внешний вид вашего стиля

Wordpress Тема имеет много преимуществ, слишком много

  • Разделяет стили и файлы шаблонов от системных файлов, поэтому сайт будет обновляться без коренных изменений визуального представления сайта
  • Разрешает кастомизацию функционала сайта , специально для конкретной темы
  • Разрешает быстрые изменения визуального отображения и макета сайта
  • Тема избавляет от нужды владельцыу сайта изучать HTML, CSS, PHP для того, чтобы сайт выглядел отлично

Почему вы должны сконструировать собственную тему WordPress ? то стоящий вопрос

  • Это возможность изучить больше CSS, HTML, PHP
  • Это возможность применить свои знания CSS, HTML, PHP в работе
  • Это креативно
  • Это весело(в большинстве случаев)
  • Если вы покажете тему сообществу, вы будете чувствовать себя замечательно, потому-что поделились чем-то с WordPress Community

Стандарты разработки тем

Wordpress темы должны разрабатываться , используя следующие стандарты :

  • Используйте хорошо-структурированный, безошибочный код PHP, HTML и CSS
  • Используйте чистый, валидный CSS
  • Следуйте направлениям дизайна тут Site Design and Layout.

Анатомия темы

WordPress тема живет в субдиректориях вордпресса (wp-content/themes/ by default), которые не могут быть перемещены , используя wp-config-file.php. Директория темы содержит файлы стилей темы, файлы шаблонов и опциональный файлы-функций(functions.php), JS- файлы и картинки. Например, тема, которая называется "test" будет находиться в каталоге wp-content/themes/test/ . Избегайте использования номера в имени темы, так как это мешает теме , находиться в списке доступных тем.

WordPress включает темы по умолчанию в каждом установочном пакете. Тщательно изучите файлы в теме по умолчанию, чтобы получить лучшее представление, о том, как построить свою файлы темы
